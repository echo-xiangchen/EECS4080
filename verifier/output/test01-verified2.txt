Formula: (((not r) and (s or q)) = (((p or (m = (i * n))) and (not (i >= m))) => (n > ((m * i) + j))))
Where: 
    p: Bool
    r: Bool
    s: Bool
    n: Int
    q: Bool
    m: Int
    j: Int
    i: Int
Is not tautology. Here is a counter example: 
    p: true
    r: true
    s: false
    n: 106
    q: false
    m: 11
    j: 10
    i: 21
